## 📊 How It Works (Simplified)

```
Real ECG Data (1000+ samples)
    ↓
[EDGE] Extract 5 Features (compression: 99%)
    ↓
[EDGE] Run AI Model Locally (<10ms)
    ↓
Decision: Normal ✓ or Abnormal ⚠️
    ↓
[IF NORMAL] Store locally (no transmission)
[IF ABNORMAL] Send small alert (128 bytes)
    ↓
[CLOUD] Backend receives alert
    ↓
[EMAIL] Doctor gets notification
```

## 🆘 Common Issues

### "Edge Impulse SDK not installed"
✅ **System automatically uses fallback mode**
- Statistical ECG analysis
- No SDK needed - fully functional
- Graceful degradation built-in

### "Cannot find backend"
✅ Start Terminal 1: `cd cloud_server && python app.py`

### "No data in dashboard"
✅ Wait 5-10 seconds for data flow
✅ Check Terminal 2 is outputting readings

### "Emails not sending"
✅ Update credentials in `email_notification.py`
✅ Use App Passwords (not regular password)
