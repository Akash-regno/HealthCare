# Smart Healthcare Monitoring Dashboard

## Description
The Smart Healthcare Monitoring Dashboard is an innovative solution designed to enable real-time monitoring and management of healthcare data. It integrates various technologies for tracking patient metrics, improving healthcare outcomes, and facilitating communication between patients and healthcare professionals.

## Features
- **Real-Time Monitoring**: Monitor vital signs and health metrics in real-time.
- **Data Visualization**: Interactive dashboards for an easy understanding of data.
- **Alerts and Notifications**: Automatic alerts for abnormal readings to ensure timely intervention.
- **User Management**: Secure login for healthcare providers and patients.
- **Reports Generation**: Generate and export reports for further analysis.

## Tech Stack
- **Frontend**: React.js, HTML5, CSS3
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT (Json Web Token)
- **Real-Time Communication**: Socket.IO

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/anjaneyasahu0502/Smart-Healthcare-Monitoring-Dashboard.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Smart-Healthcare-Monitoring-Dashboard
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- After starting the application, navigate to `http://localhost:3000` to access the dashboard.
- Follow the on-screen instructions to input your data and monitor your health metrics.
